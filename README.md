# 📷 FocusCam - Landing Page

Projeto desenvolvido com HTML e CSS simulando a página inicial de uma empresa de câmeras, com foco em layout moderno e organização visual.

---

## 🛠️ Tecnologias utilizadas
- HTML
- CSS

---

## 🎯 Objetivo
Praticar a criação de layouts completos, trabalhando navegação, posicionamento de elementos e estilização com CSS.

---

## 🌐 Visualizar projeto
https://liadaine.github.io/camera-website/

---

## 📚 O que foi praticado
- Estruturação de página com HTML  
- Criação de menu de navegação  
- Uso de background com imagem  
- Posicionamento centralizado de conteúdo  
- Estilização com CSS
